📚 BookOrbit — A Modern 3D Online Bookstore Platform

BookOrbit is a visually stunning and interactive online bookstore where authors can publish their books and users can explore, discover, and purchase them seamlessly.

✨ Features:
- 🎨 Modern UI with balanced light-mid theme
- 🧊 Immersive 3D book animations (floating, tilt, parallax effects)
- 📚 Browse books by categories, trends, and recommendations
- 🔍 Smart search and filtering
- 🛒 Add to cart & purchase flow
- ✍️ Author dashboard to upload and manage books
- 📱 Fully responsive design (mobile, tablet, desktop)

🚀 Tech Stack:
- Frontend: Next.js, Tailwind CSS
- Backend: Node.js, Express / NestJS
- Database: PostgreSQL / MongoDB
- Realtime & Animations: Framer Motion / Three.js

🌌 Vision:
To create a next-gen bookstore experience that feels alive, interactive, and futuristic with smooth 3D visuals.
